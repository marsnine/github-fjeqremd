# 재생목록 추가 모달 디자인

```
+------------------------------------------+
|  새 재생목록 추가                         X |
+------------------------------------------+
|                                          |
| [재생목록 섹션]                           |
| +--------------------------------------+ |
| | 재생목록 URL                          | |
| | +----------------+ +--------------+   | |
| | |                | |   확인        |   | |
| | +----------------+ +--------------+   | |
| |                                      | |
| | 재생목록 정보:                        | |
| | - 재생목록 이름: [자동 입력]           | |
| | - 동영상 수: [자동 입력]              | |
| | - 마지막 업데이트: [자동 입력]         | |
| +--------------------------------------+ |
|                                          |
| [채널 섹션]                              |
| +--------------------------------------+ |
| | 채널 정보:                            | |
| | - 채널 URL: [자동 입력]               | |
| | - 구독자 수: [자동 입력]              | |
| | - 채널 업데이트: [자동 입력]           | |
| +--------------------------------------+ |
|                                          |
|              [취소] [저장]               |
+------------------------------------------+

기능:
1. 재생목록 URL 입력 후 "확인" 버튼 클릭 시:
   - YouTube API로 재생목록 정보 조회
   - 재생목록 이름, 동영상 수 자동 입력
   - 채널 정보 자동 조회 및 입력

2. 저장 버튼 클릭 시:
   - 모든 정보를 external_playlists 테이블에 저장
   - 모달 닫기

3. 취소 버튼 클릭 시:
   - 입력된 정보 초기화
   - 모달 닫기
